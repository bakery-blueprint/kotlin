### Statement 과 Expression
코틀린에서 if는 식(Expression)이지 문(Statement)이 아니다. 

자바에서는 대부분이 제어 구조가 Statement 반면, 코틀린에서는 루프를 제외한 대부분이 제어 구조가 Expression 이다.
